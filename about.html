About This Site — and Its 2026 Rebuild

This site was first made in 1997, when the web was young and hand-coded HTML was just what you did. It has lived mostly unchanged since then — a personal museum, as I call it.

In early 2026 I decided to modernize it, partly because it needed it, and partly as an experiment. I wanted to test whether an AI coding agent could handle a real-world project: messy legacy HTML, inconsistent structure, no framework, no build system — just 25 years of accumulated web archaeology.

The agent I used was Claude, built by Anthropic. The process was genuinely collaborative. I described what I wanted, Claude wrote the code, I reviewed it in a local browser, gave feedback, and we iterated. Over several sessions we rebuilt the navigation, made the site responsive, restructured the i-files archive, added new content, and eventually deployed everything through a Git-based pipeline to this server.

Did it work? Better than I expected. The agent made mistakes — sometimes confidently — but it also caught things I missed, remembered context across a long project, and generally kept the work moving. The hardest parts were not the coding but the deployment plumbing: DNS, FTP, Git branch confusion, case-sensitive Linux file systems.

The result is what you see here. Same content, same yellow, same spirit — just a little more at home on a modern screen.
